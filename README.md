### Descrição do Projeto:

##### SGDB utilizado no projeto:
    MySQL Workbench v8.0.43

##### Como executar o projeto:
  Execute o **MySQL Workbench v8.0.43** e entre na conexão localhost padrão. Após carregar, vá em **"File > Open SQL Script"** e abra a pasta que está com todos os arquivos SQL extraídos.
  Assim que carregar, execute os arquivos SQL nesta ordem: **01_dll.sql**, que fará a criação do **Schema** do banco de dados e das tabelas, **02_insert.sql** que irá adicionar os dados dentro das tabelas, e por fim **03_queries.sql** que irá realizar as consultas exigidas no PDF da atividade.
